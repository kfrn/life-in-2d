# [Life in 2D](https://kfrn.github.io/life-in-2d/)

## Local setup

Dependencies:
* [Elm](https://elm-lang.org/) `0.19`
* [`create-elm-app` CLI](https://github.com/halfzebra/create-elm-app)

```bash
git clone git@github.com:kfrn/life-in-2d.git
cd life-in-2d/
elm-app start
```
